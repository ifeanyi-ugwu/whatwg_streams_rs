Streams Demos (Rust + WASM)

This site collects demos inspired by the WHATWG Streams demos and MDN Streams docs, reimplemented in Rust → WebAssembly using the whatwg_streams crate.

Available Demos

  • Append Child Writable — Custom WritableStream that appends DOM nodes (WHATWG demo).
  • Streaming Element — Stream into a custom DOM element.
  • Streaming Element with Backpressure — Variation that applies backpressure to avoid jank.
  • Streaming Fetch + Progress Bar — Fetch a large resource and stream updates into a progress bar.
  • Service Worker Streaming Response — Stream API responses through a service worker into the DOM.
  • Grayscale PNG — Fetch a PNG, convert it to grayscale via a Rust TransformStream.
  • Unpack PNG Chunks — Fetch a PNG and pipe chunks of bytes through a custom transform.

More demos coming soon — each one is isolated in <name>/.
